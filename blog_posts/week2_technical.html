Inline block vs Inline

Display Inline

When an element has an inline display that element takes up the amount of horizontal space that it's contents visually uses, assuming the element has no margin, padding or borders to account for. This means that we can have multiple elements appear on the same line and is the reason for the existence of the line break tag <br />. It is also important to note that when an element is displayed inline it will not compute padding correctly.

CSS test code
.inlinedemo {
background-color:#FF6201;
display: inline;
margin: 1em;
padding: 1em;
}
.inlineblock {
background-color:#FF6201;
display: inline-block;
margin: 1em;
padding: 1em;
}

What is Block

In CSS a block element is an element which takes up all of the horizontal space that is available to it creating a line break after the element. Some examples of this are <div>, <h1>, and <p> tags. Of course using CSS you can make any element function like a block element by simply changing its display setting for example a span.

Display Block CSS code
span {
display: block;
}

Inline Block

The idea of inline block is to create an element that will behave like an inline element where the flow of the document is concerned and behave like a block element where the inside of that element is concerned. Take these <div> elements displayed in both inline-block and inline.

As you can see the inline-block <div> handles the line break only by breaking the line for its own container whereas the inline element breaks the line for both containers.
